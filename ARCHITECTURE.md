  в”Ӯ      вҖў Validates your build
  в”Ӯ      вҖў No action needed
  в”Ӯ
  в”ңв”Җ Want to create a release for GitHub/testing?
  в”Ӯ  в””в”Җ> Create git tag вҶ’ Workflow #2 runs
  в”Ӯ      вҖў git tag v1.2.1
  в”Ӯ      вҖў git push origin v1.2.1
  в”Ӯ      вҖў Get signed APK/AAB in Releases
  в”Ӯ
  в””в”Җ Ready to publish to Play Store?

         вҖў git tag v1.3.0
         вҖў git push origin v1.3.0
         вҖў App uploads to Play Store automatically!
